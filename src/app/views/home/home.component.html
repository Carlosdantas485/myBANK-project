Bem-vindo ao MyBANK
Sobre o sistema do site
  • Este sistema de deposito foi criado com intuito de aprender a integrar o front-end com o back-end.
  • Neste projeto esta integrado com im API em Springboot no back-end.
  • Já no front-end temos o Angular consumindo a API.
  • Este projeto realiza as operações de C.R.U.D.
  • C: Create
  • R: Remove
  • U: Update
  • D: Delete
